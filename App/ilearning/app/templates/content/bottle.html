Bottle

Introdunction

Bottle is a light-weight web framework for python, and offers request dispatching (routes) with URL parameter support, templates, a built-in web server and adapters for many third-party WSGI/HTTP-server and template engines.


As a sophomore year student, you should already have learnt python last year, and able to understand the following short tutorial which introduce how to make a simple website.


Learning

Here are some very basic concept for Bottle framework:


Routing: Requests to function-call mapping with support for clean and dynamic URLs. Templates: Fast and pythonic built-in template engine and support for mako, jinja2 and cheetah templates. Utilities: Convenient access to form data, file uploads, cookies, headers and other HTTP-related metadata. Server: Built-in HTTP development server and support for paste, bjoern, gae, cherrypy or any other WSGI capable HTTP server.


Run this script or paste it into a Python console, then point your browser to http://localhost:8080/hello/world.
Now you already can make a new website! To learn more about bottle, you can access https://bottlepy.org/docs/dev/ which is the official tutorial of it.
